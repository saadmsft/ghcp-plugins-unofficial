# ralph-loop

Continuous self-referential AI loops for interactive iterative development, implementing the Ralph Wiggum technique. Run Claude in a while-true loop with the same prompt until task completion.

Ported from [anthropics/claude-plugins-official/ralph-loop](https://github.com/anthropics/claude-plugins-official/tree/main/plugins/ralph-loop).

## Prompts
- `claude-ralph-loop-cancel-ralph.prompt.md`
- `claude-ralph-loop-help.prompt.md`
- `claude-ralph-loop-ralph-loop.prompt.md`

## Install

```bash
./install.sh ralph-loop
```
