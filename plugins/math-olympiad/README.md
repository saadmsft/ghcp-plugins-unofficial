# math-olympiad

Solve competition math (IMO, Putnam, USAMO) with adversarial verification that catches what self-verification misses. Fresh-context verifiers attack proofs with specific failure patterns. Calibrated abstention over bluffing.

Ported from [anthropics/claude-plugins-official/math-olympiad](https://github.com/anthropics/claude-plugins-official/tree/main/plugins/math-olympiad).

## Skills
- `claude-math-olympiad-math-olympiad`

## Install

```bash
./install.sh math-olympiad
```
