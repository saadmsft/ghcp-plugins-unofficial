# mcp-server-dev

Skills for designing and building MCP servers that work seamlessly with Claude — guides you through deployment models (remote HTTP, MCPB, local), tool design patterns, auth, and interactive MCP apps.

Ported from [anthropics/claude-plugins-official/mcp-server-dev](https://github.com/anthropics/claude-plugins-official/tree/main/plugins/mcp-server-dev).

## Skills
- `claude-mcp-server-dev-build-mcp-app`
- `claude-mcp-server-dev-build-mcp-server`
- `claude-mcp-server-dev-build-mcpb`

## Install

```bash
./install.sh mcp-server-dev
```
