# mcp-tunnels

Connect Claude to a private MCP server through an Anthropic MCP tunnel. Drives the Docker Compose quickstart end to end: certificates, proxy config, cloudflared, and a verifiable sample server.

Ported from [anthropics/claude-plugins-official/mcp-tunnels](https://github.com/anthropics/claude-plugins-official/tree/main/plugins/mcp-tunnels).

## Prompts
- `claude-mcp-tunnels-create-docker-mcp-tunnel.prompt.md`

## Install

```bash
./install.sh mcp-tunnels
```
