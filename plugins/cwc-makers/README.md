# cwc-makers

Seamless onboarding for the Code-with-Claude Makers Cardputer: one /maker-setup command clones the build-with-claude repo, flashes UIFlow firmware, and installs the Claude Buddy app bundle onto a freshly-plugged-in M5Stack Cardputer-Adv.

Ported from [anthropics/claude-plugins-official/cwc-makers](https://github.com/anthropics/claude-plugins-official/tree/main/plugins/cwc-makers).

## Prompts
- `claude-cwc-makers-maker-setup.prompt.md`

## Skills
- `claude-cwc-makers-cardputer-buddy`
- `claude-cwc-makers-m5-onboard`

## Install

```bash
./install.sh cwc-makers
```
