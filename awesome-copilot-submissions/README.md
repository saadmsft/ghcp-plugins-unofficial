# Awesome Copilot — Plugin Submissions

This folder holds pre-drafted submission bodies for [`github/awesome-copilot`](https://github.com/github/awesome-copilot)'s [external plugin issue form](https://github.com/github/awesome-copilot/issues/new?template=external-plugin.yml).

## How to submit

1. Make sure a release tag (e.g. `v1.0.0`) exists on this repo — the form requires an **immutable ref**.
2. Open the [external plugin issue form](https://github.com/github/awesome-copilot/issues/new?template=external-plugin.yml).
3. For each plugin listed below, copy the values into the matching form field and submit.
4. Wait for `awaiting-review` → `ready-for-review` → maintainer `/approve` or `/reject`.

> Submit one at a time. Maintainers review each plugin individually.
